\# 荣耀小鸟 HonorBirdGame



\## 一、项目简介



《荣耀小鸟》是一个基于 \*\*C++ 和 SFML\*\* 开发的图形化弹弓闯关小游戏。



项目的基础玩法是：玩家通过鼠标拖拽小鸟，控制发射方向和力度，使小鸟按照抛物线飞出，撞击敌人和建筑，并通过击败全部敌人完成关卡。



在传统弹弓闯关玩法的基础上，本项目加入了 MOBA 英雄技能机制，使不同小鸟具有不同的角色定位：



\- 普通小鸟：基础发射单位；

\- 夏洛特：高速范围爆发型角色，拥有五芒星与彼岸花范围技能；

\- 花木兰：两段式技能角色，拥有轻剑突进和重剑剑气。



项目目标是将“弹弓发射玩法”“建筑破坏结构”和“英雄技能差异”结合起来，完成一个具有完整交互流程的 C++ 图形化小游戏。

\## 三、主要功能



\### 1. 基础弹弓发射



玩家可以通过鼠标拖拽小鸟来控制发射方向和力度。



小鸟发射后会按照抛物线运动，并受到重力影响。项目中使用基于 `dt` 的时间更新方式，使飞行表现更加稳定。



\### 2. 三关关卡设计



项目设计了三关关卡：



\- 第一关：基础教学关，主要用于熟悉发射、碰撞和敌人击败；

\- 第二关：加入更复杂的建筑结构，需要结合视角预览和技能使用；

\- 第三关：综合挑战关，敌人和建筑结构更加密集，对技能使用时机要求更高。



\### 3. 敌人与建筑系统



游戏中包含敌人和建筑物。小鸟撞击敌人或建筑时会造成伤害。



建筑具有生命值，受到攻击后会扣血，部分横梁结构在支撑被破坏后会下落，并可能砸中敌人。



\### 4. 英雄技能系统



\#### 夏洛特



夏洛特是范围爆发型角色。



飞行过程中按 `F` 可以释放技能，技能表现为五芒星与彼岸花绽放特效，并对范围内敌人和建筑造成伤害。



\#### 花木兰



花木兰是两段式技能角色。



\- 第一次按 `F`：轻剑突进，改变飞行状态；

\- 第二次按 `F`：短暂停顿后释放剑气，剑气按照贴图范围造成伤害。



\### 5. 视角系统



由于关卡世界宽度大于窗口显示范围，项目加入了视角控制：



\- 发射前按 `A / D` 可以左右查看关卡布局；

\- 按 `C` 可以回到发射视角；

\- 小鸟发射后，镜头会根据小鸟位置进行跟随；

\- 小鸟落地后，镜头回到发射区域。



\### 6. 中文开始界面



游戏加入了中文开始界面和玩法说明，包含：



\- 基本操作；

\- 英雄技能说明；

\- 通关目标。

\## 四、操作说明



| 操作 | 功能 |

|---|---|

| 鼠标拖拽 | 调整小鸟发射方向和力度 |

| 松开鼠标 | 发射小鸟 |

| A / D | 发射前左右查看关卡 |

| C | 回到发射视角 |

| F | 释放当前英雄技能 |

| N | 当前小鸟落地后切换下一只 |

| R | 返回主菜单 |

| Enter | 开始游戏 / 进入下一关 |
